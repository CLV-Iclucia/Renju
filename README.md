## How to build

目前只在Arch Linux和Windows下进行了简单的测试。

### Linux:

在项目根目录下使用如下命令:

```commandline
mkdir build
cd build
cmake ..
make
```

在build目录下找到Renju文件，运行即可

### Windows:

在项目根目录下使用如下命令:

```commandline
mkdir build
cd build
cmake .. -G "Unix Makefiles"
make
```

其中cmake命令与Linux下不同，避免CMake默认生成Visual Studio项目。

在build目录下找到Renju文件，运行即可。

#### 可能的问题:

如果发现界面全角字符显示错误，建议修改为UTF-8编码再运行。
